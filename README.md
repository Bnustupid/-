## 三、内容加工与整理流程
- 第一步，对照图片，查找、匹配、校对对应的文言文（简体字）（记得利用已有数据库、数字人文平台）；
- 第二步，给文本化文言文加标点（记得利用已有数字人文平台）；
- 第三步， 提取文言文中的主题词（有实际含义的词汇，例如人名、地名、时间、地点、物件等）；
- 第四步，翻译文言文成白话文
